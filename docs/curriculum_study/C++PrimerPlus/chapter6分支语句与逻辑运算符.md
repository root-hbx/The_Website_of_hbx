### （3）条件运算符（?  :  ）:
>三目运算符解析：c = a>b ? a : b
```cpp
等价于：
int c; 
if(a>b) c = a;
else c = b;
```

>通用格式：exp1 ? value1 : value2;
```cpp
eg: int c = 5>3 ? 10 : 90;    // c = 10
```
